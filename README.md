
### Set up

#### Pre req

```
cd /path/to/clone
git clone https://github.com/rrag/react-stockcharts.git

npm i
```

#### Clone the repo

```
cd /path/to/clone

git clone https://github.com/rrag/react-stockcharts-examples.git

cd react-stockcharts-examples
git submodule init
git submodule update
```

#### Install dependencies

```
npm i
```
